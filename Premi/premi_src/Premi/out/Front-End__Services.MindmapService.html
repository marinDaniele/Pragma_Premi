Class: MindmapService

Front-End::Services. MindmapService

Questa classe si occupa del recupero e della modifica delle informazioni relative ai nodi e alle associazione presenti nella mappa mentale del progetto corrente. Utilizza $http per comunicare con il back-end e MindmapAdapterService per memorizzare i dati in locale.

Methods

addAssociation(sourceNodeId, destinationNodeId) → {Promise}

Metodo che aggiunge un'associazione tra un nodo sorgente avente come id il valore del parametro sourceNodeId e un nodo destinazione avente come id il valore del parametro destinationNodeId. La creazione dell'associazione viene effettuata sia in locale sia sul back-end, utilizzando rispettivamente MindmapAdapterService e $http. Il metodo ritorna un oggetto Promise che può venire risolto o rifiutato. Nel caso la promessa venga rifiutata, verrà fornita come ragione un oggetto ErrorInfo contenente tutte le informazioni relative all’errore che si è verificato.
Parameters:
Name Type Description
sourceNodeId String Parametro che rappresenta l'id del nodo sorgente dell'associazione da creare.
destinationNodeId String Parametro che rappresenta l'id del nodo destinazione dell'associazione da creare.

addNode(parentNodeId) → {Promise}

Metodo che aggiunge un nuovo nodo alla mappa mentale del progetto corrente. Il nuovo nodo verrà aggiunto come figlio di del nodo avente id pari al valore del parametro parentId. Il metodo come prima cosa richiede la creazione del nodo al back-end, dopodiché utilizza MindmapAdapterService per aggiungere il nodo in locale. Il metodo ritorna un oggetto Promise che può venire risolto o rifiutato. Nel caso la promessa venga risolta, verrà fornito come risultato un oggetto di tipo Node contenente tutte le informazioni del nuovo nodo. Se invece la promessa viene rifiutata, sarà fornita come ragione un oggetto ErrorInfo contenente tutte le informazioni relative all’errore che si è verificato.
Parameters:
Name Type Description
parentNodeId String Parametro che rappresenta l'id del nodo padre a cui aggiungere il nuovo nodo figlio.

getAssociableNodeList(nodeId) → {Object}

Metodo che dato l'id di un nodo fornisce l'insieme dei nodi che possono essere associati ad esso. Tra due nodi può essere definita una sola associazione e, se tra un nodo e l'altro è già presente una relazione gerarchica, non è possibile aggiungere un'associazione. Le informazioni dei nodi sono memorizzate all'interno dell'oggetto ritornato sotto forma di coppie chiave/valore, usando come chiave l'id del nodo associabile e come valore il titolo del nodo.
Parameters:

nodeId String Parametro che rappresenta l'id del nodo per il quale si vuole recuperare la lista dei nodi associabili.
