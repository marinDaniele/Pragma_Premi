Metodo statico che costruisce un nuovo progetto che conterrà il primo nodo vuoto della mappa mentale. Verrà inoltre istanziato il percorso di default. Restituisce un oggetto JSON che descrive l’elemento oppure un messaggio di errore.

removeNode(nodeId, callback, errback)

Metodo che consente di rimuovere nodi dal progetto e quindi dalla mappa mentale. Se un nodo da rimuovere è radice di un sottoalbero, devono essere rimossi anche tutti i figli e le relazioni che coinvolgono il nodo. Deve aggiornare anche tutti i percorsi di presentazione che contenevano quel nodo. Restituisce un oggetto JSON che descrive la mappa mentale priva del sottoalbero rimosso oppure un messaggio di errore.
Parameters:
Name Type Description
nodeId ObjectId Identificativo del nodo da rimuovere.
callback function Rappresenta la callback che il metodo deve chiamare al termine dell’elaborazione nel caso in cui non si siano verificati errori.
errback function Rappresenta la callback che il metodo deve chiamare qualora si verificassero errori durante l’esecuzione del metodo.
