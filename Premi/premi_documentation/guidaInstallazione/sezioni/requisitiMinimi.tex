\section{Requisiti di sistema}\label{requisiti}
Per poter eseguire il \gloxy{server} dell'applicativo Ã¨ necessaria una macchina che consenta di installare ed eseguire Node.js e \gloxy{MongoDB}. In particolare, i sistemi operativi sicuramente supportati sono:
\begin{itemize}
\item \textit{Microsoft Windows}, dalla versione 2008 \gloxy{Server} in poi;
\item \textit{Ubuntu o derivate}, dalla versione 12.04 LTS 64-bit in poi;
\item \textit{Mac OSX}, dalla versione 10.7 64-bit in poi.
\end{itemize}
Per quanto riguarda i requisiti minimi \gloxy{hardware}, si richiedono:
\begin{itemize}
\item spazio libero su disco di almeno 1GB;
\item 1GB di memoria RAM totale;
\item 500MB di memoria RAM disponibile;
\item processore core singolo da almeno 2.0GHz.
\end{itemize}
